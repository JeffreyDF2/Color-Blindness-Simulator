As you could see, individuals with colorblindness can have a very difficult time contrasting colors while those without wouldn't have any difficulties. The internet should be a place for everyone to be able to enjoy, thus there should be ways to improve the experience for those with colorblindness. Many videogames do this by offering color filters for certain color blindness. Another way to help with this problem is to create a browser extension (something similar to honey) that can able a color filter onto any website, helping the user differentiate between difficult colors.
